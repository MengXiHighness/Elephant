Install:
1) Run Generator-65.msi to install CodeSmith Generator v6.5.0.14588
2) Run CodeSmith.Licensing.v6.5.exe and click "Install" button

Uninstall:
1) Run CodeSmith.Licensing.v6.5.exe and click "Uninstall" button

Notes:
1) CodeSmith.Licensing.v6.5.exe can only patch and license CodeSmith Generator v6.5.0.14588
2) Close Visual Studio 2010/2011 and programs of CodeSmith Generator before running CodeSmith.Licensing.v6.5.exe
3) The Install of CodeSmith.Licensing.v6.5.exe is patching and licensing CodeSmith Generator v6.5.0.14588
4) The Uninstall of CodeSmith.Licensing.v6.5.exe is restoring CodeSmith Generator v6.5.0.14588 to its original state
